# 你不会百度吗
该项目为 nbhbdm.cn 的开发项目。   
部分代码基于 [wolfbolin/baidux](https://github.com/wolfbolin/baidux)。   
## 使用的 JavaScript 库
* Vue.js
* jQuery
* Clipboard.js

## 开源声明
我们向您开放该网站的所有代码。您可以按照合理的方式对该代码进行使用。   
但您不能使用这些内容：
* 「你不会百度吗」Logo
* 网页中的 Matomo 统计代码
