# 你不会Microsoft Bing吗
该项目为 nbhbingm.tk 的开发项目。   
部分代码基于 [wolfbolin/baidux](https://github.com/wolfbolin/baidux)和[TheZihanGu/nbhbdm](https://github.com/TheZihanGu/nbhbdm)。   

## 使用的 JavaScript 库
* Vue.js
* jQuery
* Clipboard.js
